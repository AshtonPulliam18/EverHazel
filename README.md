# EverHazel

EverHazel is a personal, independent project I am building outside of my day job. It is also a real business in progress - a premium, conversion-focused website and lead-capture system for med spa growth.

## Highlights
- Conversion-focused landing experience
- Clean, minimal design with moss-green brand accents
- Contact flow with automated confirmation email
- Calendly booking CTA in a modal
- Mobile-first layout and fast performance

## Testing
I plan to add Playwright for end-to-end testing to validate lead flow and booking UX.

## Tech
- Codex
- Next.js (App Router)
- React
- Tailwind CSS
- Resend (transactional email)
